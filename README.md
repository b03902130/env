# Linux environment

## Install
1. clone down this vim repository
2. `cd env`
3. `source install.sh`
4. all done

## Upgrade
1. `source upgrade.sh`

## Uninstall
1. `source uninstall.sh`

## Note
- vim-fugitive is now working inproperly with Glog, one can copy the `vim-fugitive/` provided into ~/.vim/bundle/ to solve the issue
